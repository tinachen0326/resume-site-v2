前言
過去的工作經驗著重在於視覺設計，長期擔任電商產業的美編設計人員，對產業的運作模式與行銷邏輯具有一定理解。在過去工作經驗中，我逐漸關注使用者在網頁中的瀏覽行為，進而擁有以使用者為核心的設計思考。這段經歷不僅讓我具有數位行銷的思考，也讓我從單純視覺表現延伸至整體使用者體驗的設計思維。
轉職動機
過去的工作歷練，讓我擅長設計排版與行銷素材設計。但我發現設計職位缺乏長期策略的投入，這促使我重新思考職涯方向，並期望能朝向結合使用者需求、產品價值與技術實現的 UI/UX 或網頁設計領域發展，建立更具延續性的專業角色。
學習過程／專案經驗
為了實現轉職目標，我參與產業新尖兵系統性的課程，學習前端（HTML、CSS、JavaScript、Bootstrap、Vue.js、React）與後端（Python Django、PHP Laravel）等技術，並實作多項整合 UI 設計與工程實作的專案。從中我更明確地發現到自己在UI/UX 設計流程、使用者行為分析、界面邏輯與畫面規劃等方面具有高度興趣與能力。
未來展望
我希望未來能投身於一個結合創造力與技術的產品團隊，發揮我在 UI 設計、使用者體驗、跨部門溝通與前端整合的能力。也期望持續精進更多程式語言與框架，成為具備全方位產品思維、能夠整合設計與技術的 UI/UX 設計師，成為團隊中不可或缺的一員。
